# MoneaTracker
> 몸과 마음의 리듬을 기록하는 감성 생리주기 캘린더 앱
MoneaTracker는 생리 주기, 배란일, 가임기, 감정 변화 등을  
직관적인 다크모드 UI로 관리할 수 있는 헬스케어 앱입니다.
---
## ✨ Features
- 생리 주기 계산
- 배란일 / 가임기 예측
- 월별 캘린더
- 감정 & 증상 기록
- 통계 및 패턴 분석
- 알림 기능
---
## 🎨 Design
- Dark Mode

- Minimal UI
- Emotional Tracking
